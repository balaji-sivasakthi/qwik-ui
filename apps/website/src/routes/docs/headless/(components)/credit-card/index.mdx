# Credit Card

#### The CreditCard component is a reusable UI component that allows users to input credit card information, including the card holder name, card number, expiry date, and CVV (Card Verification Value).

{' '}

<MainExample>
  ```tsx
  <CreditCard handleSubmit={handleSubmit} />
  ```
</MainExample>
